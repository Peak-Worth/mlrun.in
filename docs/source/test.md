<h2 id="2-emphasis">2. Emphasis</h2>
<p>Emphasis, aka italics, with <em>asterisks</em> or <em>underscores</em>. {<code>*Text*</code>}</p>
<p>Strong emphasis, aka bold, with <strong>asterisks</strong> or <strong>underscores</strong>. {<code>**Text**</code> or <code>__Text__</code>}</p>
<p>Combined emphasis with <strong>asterisks and <em>underscores</em></strong>. {<code>**bold and _italics_**</code>}</p>
<p>Strikethrough uses two tildes. <s>Scratch this.</s> {<code>~~Text~~</code>}</p>
<h2 id="3-lists">3. Lists</h2>
<p>Unordered list can use</p>
<ul>
<li>asterisks {*}</li>
</ul>
<ul>
<li>Or minuses {-}</li>
</ul>
<ul>
<li>Or pluses {+}</li>
</ul>
<p>For ordered lists use numbers</p>
<ol>
<li>First ordered list item</li>
<li>Another item</li>
</ol>
